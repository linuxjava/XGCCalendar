# XGCCalendar

- 支持矩形和圆角矩形效果；
- 支持设置padding扩大触摸区域，确保滑动效果更好；

## APK下载
[Download](https://github.com/linuxjava/XGCIndexBar/raw/master/apk/app-debug.apk)

## 使用
### 添加依赖
```xml
implementation 'xgc.free.indexbar:IndexBar:0.1'
```
